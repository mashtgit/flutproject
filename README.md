# flutter app:

## possibilities

1. **Беседа**: Realtime Speech-to-Speech Translation App - VAD (Voice Activity Detection), обеспечивающее двусторонний, синхронный перевод речи с минимальной задержкой.

- Выбор двух языков: **L1 ↔ L2**
- Пользователь говорит на L1.
- app переводит и озвучивает на L2.
- app переводит и озвучивает L2 и любой язык на L1.

2. **Текстовый**: Перевод введенного текста

- пользователь вводит текст в поле, нажимает на значок "translate", 
  текст переводится на язык L2,
  значок "to speech" - озвучивает переведенный текст на язык L2.

  ## safety:
- Ключи доступа к Google Cloud будут храниться только внутри Cloud Function, не в приложении.
