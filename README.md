# flutter app:

## possibilities

### режим "gemini":
**обычное общение пользователя с gemin.**

- текстовое  поле для пользовательсково промта + значок "send".
- текстовое  поле для получение ответов от гемини
- значок микрофона - `пользователь говорит` → `gemini` → `отвечает голосом` → `пользователь слышит`.

### режим "dialogue":
**распознование, перевод и озвучка речи в реальном времени.**

- пользователь выбирает свой язык `L1` и язык опонента `L2`.
пользователь нажимает  на значок микрофона → `сессия` начинается.
- `сессия`: `app слушает`, `распознаёт человеческие речи`, `языки` → gemini обработывает с помощью systemPromt: "Ты — профессиональный синхронный переводчик. озвучаи любой речь на язык L1, Если слышишь L1 - озвучивай на L2.
Не добавляй никаких объяснений и фраз от себя, только перевод. Используй естественный, разговорный стиль."

### 
возможность захвата изображений из камеры.
- gemini анализирует изображение, если на изображении есть текст: гемини переводить ево на язык L1. 
- если на изображении нету текста: Ggemini делает короткую описание сцены.

переведенный текст или описание отоброжаетса в низу изображений.
